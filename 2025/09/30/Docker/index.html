真正的示例,隔离网络,文件.不同容器可能会争抢资源1

docker ps

docekr start/stop/rm ID
docker commit -a “作者名称” -m “log信息” ID 打包成镜像
docker cp 文件目录 容器ID:目标目录 从前拷贝到后
docker exec -it ID /bin/bash 进入容器内部
